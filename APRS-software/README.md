# APRS-software

- KISS_class.ino

	- RX and TX APRS LoRa with Kiss data protocol on USB serial 9600 bauds 

- LoRaReceiver_simple.ino

	- Just a LoRa receiver. Print raw data on USB serial 
	
- TX_APRS.ino

	- Send LoRa APRS Position, message and Weather
